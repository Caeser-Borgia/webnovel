【快速开始】

1. 双击 start.bat 启动
2. 自动打开浏览器（http://localhost:3000）
3. 输入 API Key 和大模型地址
4. 开始创作！

【系统要求】

- Windows 7 及以上
- 需要安装 Node.js 18+ 版本（如未安装，会有提示下载链接）
- 网络连接（用于调用大模型 API）

【使用步骤】

第一步：配置 API
  - 选择大模型：DeepSeek / Claude / 通义千问 等任意兼容 OpenAI 的 API
  - 填入 API 地址（例：https://api.deepseek.com/chat/completions）
  - 填入 API Key（从大模型平台获取）
  - 选择小说风格：网文 / 科幻 / 言情 / 悬疑

第二步：设置故事
  - 输入书名（例：我穿越了，但不想修仙）
  - 输入主角名字（例：李火旺）
  - 填写基本设定（世界观、背景等）

第三步：创作生成
  - 点击"开始创作"
  - 实时查看小说生成过程
  - 可以暂停、重新开始、复制或下载

【API 配置指南】

【DeepSeek】（推荐新手）
  - 网址：https://platform.deepseek.com
  - API 地址：https://api.deepseek.com/chat/completions
  - 模型名称：deepseek-chat
  - 优点：便宜、快速

【Claude 3.5 Sonnet】（推荐高质量）
  - 网址：https://console.anthropic.com
  - API 地址：https://api.anthropic.com/v1/messages
  - 模型名称：claude-3-5-sonnet-20241022
  - 优点：质量最好，但需要代理支持

【通义千问】
  - 网址：https://bailian.console.aliyun.com
  - API 地址：https://dashscope.aliyuncs.com/api/v1/services/aigc/text-generation/generation
  - 模型名称：qwen-max

【常见问题】

Q：运行后提示"找不到 Node.js"怎么办？
A：请安装 Node.js 18+ 版本，下载地址：https://nodejs.org/

Q：浏览器一直不打开怎么办？
A：可能是网络问题，请手动访问 http://localhost:3000

Q：能生成多长的小说？
A：理论上无限长，每次可生成 3000-5000 字，持续点击"继续生成"即可累积

Q：生成效果不好？
A：
  - 提高故事设定的详尽程度
  - 尝试切换大模型（Claude 效果最好）
  - 调整提示词（可在后续版本支持自定义）

Q：怎么导出成 Word 或 PDF？
A：生成完后可直接下载 DOCX 格式，用 Word 打开即可

【故障排除】

症状：点击"开始创作"没反应
解决：
  1. 检查网络连接
  2. 检查 API Key 是否正确
  3. 查看浏览器控制台是否有错误（按 F12）
  4. 重启浏览器和应用

症状：服务器启动后浏览器没打开
解决：手动在浏览器地址栏输入 http://localhost:3000
